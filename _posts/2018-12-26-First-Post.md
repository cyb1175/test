# I. 스펙(Specification)

구현된 어플리케이션의 주요 기능

동영상 검색 기능
  * 사용자가 입력을 통해 챗봇하게 전송한 데이터를 분석한다.
  * 입력받은 데이터를 바탕으로 동영상을 검색한다

# II. 회고(Retrospective)

어플리케이션 구현 과정에서의 어려움과 문제점
  * 데이터를 크롤링 할 때 영상은 바로 가져오기 어렵기 때문에 힘들었다.
  
# III. 보완 계획(Feedback)

현재 미완성이지만 추가로 구현할 기능 및 기존 문제점 보완 계획

  * 영상 썸네일 연결 기능
